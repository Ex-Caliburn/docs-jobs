# 接口文档
#### 1.兼职列表
> 可添加的课程列表

**请求地址：**

**前端请求地址：**../IntegralMall/GetIntegralCourseList

**请求方式：**GET

**参数：**

| 参数 | 必选 | 类型 | 参数说明 |
| ---- | ---- | ---- | -------- |
| offset         | 否   | int    | 分页偏移     |
| limit          | 否   | int    | 分页大小     |

**返回：**

```json
{
  "data": {
    "count": 3,
    phone: '',
    info: '',
    about: '', 
  },
  "err_code": 0,
  "err_msg": "success"
}
